# 🌍 Volta ao Mundo - Itália

Projeto desenvolvido para apresentação acadêmica utilizando **HTML, CSS, Javascript e Bootstrap**.

O site tem como objetivo apresentar informações sobre a Itália, incluindo cidades, comidas típicas e pontos turísticos, com um layout moderno, responsivo e organizado.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Javascript
- Bootstrap

---

## 📂 Estrutura do Projeto

```
/Volta-ao-Mundo
│
├── index.html        (Página principal)
├── cidades.html      (Cidades da Itália)
├── comidas.html      (Comidas típicas)
├── turismo.html      (Pontos turísticos)
├── termos.html       (Termos e política)
├── style.css         (Estilização personalizada)
```

---

## 🎯 Funcionalidades

- Layout totalmente responsivo 📱💻
- Navegação entre páginas 🔗
- Carrossel interativo 🎠
- Imagens e vídeos integrados 🎥
- Design com cores da bandeira italiana
